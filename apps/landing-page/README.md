# Cortex landing page

TanStack Start, static-prerendered. The split-flap hero on carbon.

```bash
npm run dev -w @animus-labs/landing-page      # local dev on :3000
npm run build -w @animus-labs/landing-page    # static prerender to dist
```

Brand colors, type, and the split-flap primitives come from `@animus-labs/brand`.
The board animation itself lives here in `src/components/SplitFlapBoard.tsx`.
